# Game-_-Ordering-numbers
숫자 퍼즐을 순서대로 맞추는 게임입니다.
[20]    [15]    [07]    [04]    [19]
[11]    [14]    [23]    [09]    [10]
[18]    [16]    [12]    [08]    [06]
[05]    [21]    [13]    [01]    [22]
[02]    [03]    [17]    [ *]    [24]
w : 위 // s : 아래 // a : 왼쪽 // d : 오른쪽 // q : 종료
예를 들면 이런 화면을 처음 출력해주고
키보드를 이용하여 *를 옮겨가며 숫자를 순서대로 맞추면 되는 게임입니다.
